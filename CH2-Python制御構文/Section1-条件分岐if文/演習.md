## 演習2: はじめての if 文

### 問題

年齢が 20 以上なら「成人です」と表示するプログラムを作成してください。

## 演習3: if-else 文

### 問題

数値が正の数か負の数かを判定するプログラムを作成してください。

```python
number = -5
```

- 正の数なら「正の数です」
- そうでなければ「負の数または0です」

### 解答例

```python
number = -5

if number > 0:
    print("正の数です")
else:
    print("負の数または0です")
```

## 演習4: if-elif-else 文

### 問題

点数に応じて成績を表示するプログラムを作成してください。

| 点数 | 成績 |
|------|------|
| 90以上 | S |
| 80以上 | A |
| 70以上 | B |
| 60以上 | C |
| 60未満 | F |

```python
score = 78
```

### 解答例

```python
score = 78

if score >= 90:
    print("成績: S")
elif score >= 80:
    print("成績: A")
elif score >= 70:
    print("成績: B")
elif score >= 60:
    print("成績: C")
else:
    print("成績: F")
```

### 確認ポイント
- [ ] `score = 78` のとき「成績: B」と表示されましたか？
- [ ] いくつかの値で試してみましたか？

## 演習5: and 演算子

### 問題

以下の条件を満たす場合に「入場可能」と表示してください：
- 年齢が 18 以上
- チケットを持っている（has_ticket が True）

```python
age = 20
has_ticket = True
```

### 解答例

```python
age = 20
has_ticket = True

if age >= 18 and has_ticket:
    print("入場可能")
else:
    print("入場できません")
```

## 演習8: 偶数・奇数の判定

### 問題

数値が偶数か奇数かを判定するプログラムを作成してください。

**ヒント**: 偶数は 2 で割った余りが 0

```python
number = 7
```

### 解答例

```python
number = 7

if number % 2 == 0:
    print(str(number) + "は偶数です")
else:
    print(str(number) + "は奇数です")
```

## 演習9: 文字列の比較

### 問題

パスワードが正しいかチェックするプログラムを作成してください。

- 正しいパスワード: `"python123"`
- 一致したら「ログイン成功」
- 一致しなければ「パスワードが違います」

```python
password = "python123"
```

### 解答例

```python
password = "python123"
correct_password = "python123"

if password == correct_password:
    print("ログイン成功")
else:
    print("パスワードが違います")
```

## 演習11: 複数条件の組み合わせ

### 問題

以下の条件で映画のチケット料金を判定してください：

| 条件 | 料金 |
|------|------|
| 12歳以下 | 800円 |
| 65歳以上 | 1000円 |
| それ以外 | 1800円 |

```python
age = 70
```

### 解答例

```python
age = 70

if age <= 12:
    print("料金: 800円（子供）")
elif age >= 65:
    print("料金: 1000円（シニア）")
else:
    print("料金: 1800円（一般）")
```

## 演習12: ネストした if 文

### 問題

以下の条件で処理を分岐させてください：

1. まず性別（"男性" or "女性"）で分岐
2. 次に年齢で分岐（20歳以上かどうか）

```python
gender = "男性"
age = 25
```

### 解答例

```python
gender = "男性"
age = 25

if gender == "男性":
    print("性別: 男性")
    if age >= 20:
        print("成人男性です")
    else:
        print("未成年男性です")
else:
    print("性別: 女性")
    if age >= 20:
        print("成人女性です")
    else:
        print("未成年女性です")
```
