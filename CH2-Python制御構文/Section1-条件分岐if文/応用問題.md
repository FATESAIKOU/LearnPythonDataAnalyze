## 応用問題1: じゃんけんゲーム

### 問題

ユーザーの入力（グー、チョキ、パー）に対して、コンピュータの手を「グー」に固定し、勝敗を判定するプログラムを作成してください。

### 出力例

```
あなたの手を入力（グー/チョキ/パー）: チョキ
コンピュータの手: グー
結果: あなたの負け！
```

### ヒント

- じゃんけんのルール
  - グー vs チョキ → グーの勝ち
  - チョキ vs パー → チョキの勝ち
  - パー vs グー → パーの勝ち

<details>
<summary>解答例を見る</summary>

```python
player = input("あなたの手を入力（グー/チョキ/パー）: ")
computer = "グー"

print("コンピュータの手: " + computer)

if player == computer:
    print("結果: あいこ！")
elif player == "グー":
    print("結果: あいこ！")
elif player == "チョキ":
    print("結果: あなたの負け！")
elif player == "パー":
    print("結果: あなたの勝ち！")
else:
    print("正しい手を入力してください")
```

</details>

## 応用問題2: 電気料金計算

### 問題

使用量に応じて電気料金を計算するプログラムを作成してください。

| 使用量 | 料金 |
|--------|------|
| 0〜120kWh | 1kWh あたり 20円 |
| 121〜300kWh | 最初の120kWhは20円、残りは25円 |
| 301kWh以上 | 最初の120kWhは20円、次の180kWhは25円、残りは30円 |

### 出力例

```
使用量（kWh）: 350
================================
使用量: 350kWh
基本料金部分（0-120kWh）: 2400円
中間料金部分（121-300kWh）: 4500円
高額料金部分（301kWh-）: 1500円
--------------------------------
合計: 8400円
================================
```

<details>
<summary>解答例を見る</summary>

```python
usage = int(input("使用量（kWh）: "))

print("================================")
print("使用量: " + str(usage) + "kWh")

if usage <= 120:
    total = usage * 20
    print("基本料金部分（0-120kWh）: " + str(total) + "円")
elif usage <= 300:
    base = 120 * 20
    mid = (usage - 120) * 25
    total = base + mid
    print("基本料金部分（0-120kWh）: " + str(base) + "円")
    print("中間料金部分（121-300kWh）: " + str(mid) + "円")
else:
    base = 120 * 20
    mid = 180 * 25
    high = (usage - 300) * 30
    total = base + mid + high
    print("基本料金部分（0-120kWh）: " + str(base) + "円")
    print("中間料金部分（121-300kWh）: " + str(mid) + "円")
    print("高額料金部分（301kWh-）: " + str(high) + "円")

print("--------------------------------")
print("合計: " + str(total) + "円")
print("================================")
```

</details>

## 応用問題3: うるう年判定

### 問題

入力された年がうるう年かどうかを判定するプログラムを作成してください。

**うるう年の条件:**
1. 4で割り切れる年はうるう年
2. ただし、100で割り切れる年はうるう年ではない
3. ただし、400で割り切れる年はうるう年

### 出力例

```
年を入力: 2024
2024年はうるう年です
```

```
年を入力: 1900
1900年はうるう年ではありません
```

<details>
<summary>解答例を見る</summary>

```python
year = int(input("年を入力: "))

if year % 400 == 0:
    print(str(year) + "年はうるう年です")
elif year % 100 == 0:
    print(str(year) + "年はうるう年ではありません")
elif year % 4 == 0:
    print(str(year) + "年はうるう年です")
else:
    print(str(year) + "年はうるう年ではありません")
```

</details>

## 応用問題5: 成績評価システム

### 問題

3つの科目の点数を入力し、平均点と総合評価を表示するプログラムを作成してください。

**評価基準:**
| 平均点 | 評価 | コメント |
|--------|------|----------|
| 90以上 | A | 素晴らしい！ |
| 80以上 | B | よくできました！ |
| 70以上 | C | 合格です |
| 60以上 | D | もう少し頑張りましょう |
| 60未満 | F | 再試験が必要です |

### 出力例

```
国語の点数: 85
数学の点数: 78
英語の点数: 92
================================
    成績表
================================
国語: 85点
数学: 78点
英語: 92点
--------------------------------
合計: 255点
平均: 85.0点
--------------------------------
評価: B
よくできました！
================================
```

<details>
<summary>解答例を見る</summary>

```python
japanese = int(input("国語の点数: "))
math = int(input("数学の点数: "))
english = int(input("英語の点数: "))

total = japanese + math + english
average = total / 3

print("================================")
print("    成績表")
print("================================")
print("国語: " + str(japanese) + "点")
print("数学: " + str(math) + "点")
print("英語: " + str(english) + "点")
print("--------------------------------")
print("合計: " + str(total) + "点")
print("平均: " + str(average) + "点")
print("--------------------------------")

if average >= 90:
    print("評価: A")
    print("素晴らしい！")
elif average >= 80:
    print("評価: B")
    print("よくできました！")
elif average >= 70:
    print("評価: C")
    print("合格です")
elif average >= 60:
    print("評価: D")
    print("もう少し頑張りましょう")
else:
    print("評価: F")
    print("再試験が必要です")

print("================================")
```

## 応用問題6: 自動販売機シミュレーション

### 問題

自動販売機のシミュレーションプログラムを作成してください。

**商品リスト:**
| 番号 | 商品 | 価格 |
|------|------|------|
| 1 | コーラ | 120円 |
| 2 | お茶 | 100円 |
| 3 | コーヒー | 130円 |
| 4 | 水 | 80円 |

**仕様:**
1. 投入金額を入力
2. 商品番号を入力
3. 購入可能かチェック
4. お釣りを計算して表示

### 出力例

```
===== 自動販売機 =====
1. コーラ (120円)
2. お茶 (100円)
3. コーヒー (130円)
4. 水 (80円)
=======================
投入金額: 200
商品番号を選択: 3
-----------------------
コーヒーを購入しました！
お釣り: 70円
=======================
```

<details>
<summary>解答例を見る</summary>

```python
print("===== 自動販売機 =====")
print("1. コーラ (120円)")
print("2. お茶 (100円)")
print("3. コーヒー (130円)")
print("4. 水 (80円)")
print("=======================")

money = int(input("投入金額: "))
choice = int(input("商品番号を選択: "))

print("-----------------------")

if choice == 1:
    item = "コーラ"
    price = 120
elif choice == 2:
    item = "お茶"
    price = 100
elif choice == 3:
    item = "コーヒー"
    price = 130
elif choice == 4:
    item = "水"
    price = 80
else:
    item = ""
    price = 0

if item == "":
    print("無効な商品番号です")
elif money < price:
    print("お金が足りません")
    print("必要金額: " + str(price) + "円")
    print("不足金額: " + str(price - money) + "円")
else:
    print(item + "を購入しました！")
    change = money - price
    print("お釣り: " + str(change) + "円")

print("=======================")
```

</details>

## 🏆 チャレンジ問題

### 問題: 数当てゲームのヒント機能

コンピュータが決めた数字（1〜100）をユーザーが当てるゲームの判定部分を作成してください。

- 正解なら「正解！」
- 入力が正解より小さければ「もっと大きい数字です」
- 入力が正解より大きければ「もっと小さい数字です」
- 入力が範囲外（1〜100以外）なら「1〜100の範囲で入力してください」

**ヒント**: 繰り返しは次のセクションで学ぶので、1回の判定だけで OK

<details>
<summary>解答例を見る</summary>

```python
answer = 42  # 正解の数字（実際はランダムにする）
guess = int(input("1〜100の数字を入力: "))

if guess < 1 or guess > 100:
    print("1〜100の範囲で入力してください")
elif guess == answer:
    print("正解！")
elif guess < answer:
    print("もっと大きい数字です")
else:
    print("もっと小さい数字です")
```

</details>
