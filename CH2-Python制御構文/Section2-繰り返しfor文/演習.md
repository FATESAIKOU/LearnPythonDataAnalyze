## 演習7: リストの合計

### 問題

以下のリストの合計を計算してください。

```python
numbers = [5, 10, 15, 20, 25]
```

### 解答例

```python
numbers = [5, 10, 15, 20, 25]
total = 0

for num in numbers:
    total = total + num

print("合計: " + str(total))
```

### 期待される出力

```

```

## 演習8: enumerate() の使用

### 問題

以下のリストを「番号: 要素」の形式で表示してください（番号は1から）。

```python
animals = ["犬", "猫", "鳥", "魚"]
```

### 期待される出力

```
1: 犬
2: 猫
3: 鳥
4: 魚
```

### 解答例

```python
animals = ["犬", "猫", "鳥", "魚"]

for index, animal in enumerate(animals, 1):
    print(str(index) + ": " + animal)
```

## 演習14: 条件に合う要素をカウント

### 問題

以下のリストで、50以上の数をカウントしてください。

```python
scores = [45, 78, 32, 90, 55, 88, 42, 67]
```

### 解答例

```python
scores = [45, 78, 32, 90, 55, 88, 42, 67]
count = 0

for score in scores:
    if score >= 50:
        count = count + 1

print("50以上の数: " + str(count) + "個")
```

### 期待される出力

```
50以上の数: 5個
```

## 演習15: ネストした for ループ

### 問題

以下のパターンを表示してください。

```
1-1 1-2 1-3 
2-1 2-2 2-3 
3-1 3-2 3-3 
```

### ヒント

- 外側のループで行（1〜3）
- 内側のループで列（1〜3）
- 同じ行に表示するには `print(..., end=" ")` を使う

### 解答例

```python
for i in range(1, 4):
    for j in range(1, 4):
        print(str(i) + "-" + str(j), end=" ")
    print()  # 改行
```
