# CH2: Python 制御構文 - プログラムの流れを制御する

## 🎯 この章の目的

プログラムの流れを自在にコントロールできるようになることを目指します。
条件によって処理を変えたり、同じ処理を繰り返したりする方法を学び、より実用的なプログラムが書けるようになります。

---

## 📚 教授技術

| セクション | トピック | 学習内容 |
|------------|----------|----------|
| Section1 | 条件分岐（if文） | if, elif, else を使った条件分岐 |
| Section2 | 繰り返し（for文） | for ループとイテラブル、range() |
| Section3 | 繰り返し（while文） | while ループ、break と continue |

---

## 🎓 教授方法

### アプローチ
- **図解で理解**: フローチャートでプログラムの流れを視覚化
- **パターンで覚える**: よく使う書き方をパターンとして習得
- **段階的に複雑に**: 単純な例から始めて徐々にネストを増やす

### 進め方
1. 各セクションの「技術説明.md」を読む
2. フローチャートを見てプログラムの流れをイメージ
3. コードを実行して動作を確認
4. 「演習.md」で手を動かして定着
5. 「応用問題.md」で実践力を養う

## 📖 セクション構成

### Section1: 条件分岐（if文）
**目標**: 条件によって異なる処理を実行できるようになる

- 比較演算子（==, !=, <, >, <=, >=）
- 論理演算子（and, or, not）
- if 文の基本構文
- if-else 文
- if-elif-else 文
- 条件のネスト

### Section2: 繰り返し（for文）
**目標**: リストや範囲に対して繰り返し処理ができる

- for 文の基本構文
- range() 関数の使い方
- リストの要素を順番に処理
- enumerate() で添え字も取得
- ネストした for ループ

### Section3: 繰り返し（while文）
**目標**: 条件が満たされる間、処理を繰り返せる

- while 文の基本構文
- 無限ループと break
- continue でスキップ
- while-else 構文
- for と while の使い分け

## 💡 つまずきやすいポイント

| 問題 | 解決策 |
|------|--------|
| インデントエラー | Python はインデント（空白4つ）が重要。揃えているか確認 |
| 条件が正しく判定されない | == と = の違いを確認（== は比較、= は代入） |
| 無限ループ | while の条件が必ず False になるか確認 |
| range の範囲がずれる | range(n) は 0 から n-1 まで。range(1, n+1) で 1 から n |
