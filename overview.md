# Python データ分析入門 - 教材概要

## 🎯 教材の目的

本教材は、プログラミング未経験者が **Python を使ったデータ分析** の基礎を短期間で習得することを目的としています。AI エージェントがあなたの学習をサポートし、つまずきやすいポイントを丁寧に解説します。

---

## 📚 教授技術

本教材では以下の技術を段階的に学習します：

| 技術 | 概要 | 学習目標 |
|------|------|----------|
| **Python 基礎** | プログラミングの基本文法 | 変数、データ型、制御構文、関数を理解する |
| **NumPy** | 数値計算ライブラリ | 配列操作と数値計算の基礎を習得する |
| **Pandas** | データ操作ライブラリ | データの読み込み、加工、集計ができるようになる |
| **Matplotlib** | データ可視化ライブラリ | グラフ作成でデータを視覚的に表現できるようになる |

---

## 🎓 教授方法

### 学習アプローチ
- **スモールステップ方式**: 小さな成功体験を積み重ねて、達成感を得ながら学習
- **ハンズオン重視**: 説明を読むだけでなく、実際にコードを動かして理解を深める
- **AI エージェントサポート**: 分からないことはいつでも AI に質問できる環境

### 各セクションの構成
1. **技術説明.md** - 概念と基本的な使い方を解説
2. **演習.md** - 手を動かして学ぶ練習問題
3. **応用問題.md** - 実践的な課題にチャレンジ

### 学習環境
- **Jupyter Notebook**: ブラウザ上でコードを書いて即座に結果を確認
- **Docker**: どの OS でも同じ環境で学習可能

## 📖 章構成

### CH1: Python 基礎 - はじめの一歩
プログラミングの世界へようこそ！変数、データ型、基本的な操作を学びます。

### CH2: Python 制御構文 - プログラムの流れを制御する
条件分岐（if文）とループ（for/while）を使って、プログラムを自在に操ります。

### CH3: Python 関数とモジュール - コードを整理する
再利用可能なコードの書き方と、外部ライブラリの使い方を学びます。

### CH4: NumPy 入門 - 数値計算の基礎
配列（array）の作成と操作、基本的な数値計算を習得します。

### CH5: Pandas 入門 - データ操作の基礎
DataFrame を使ったデータの読み込み、選択、フィルタリングを学びます。

### CH6: Pandas 応用 - データの加工と集計
データのクリーニング、変換、グループ化、集計を実践します。

### CH7: Matplotlib 入門 - データの可視化
折れ線グラフ、棒グラフ、散布図など、基本的なグラフを作成します。

### CH8: 総合演習 - ミニプロジェクト
これまで学んだ技術を組み合わせて、実際のデータ分析プロジェクトに挑戦します。

---

## 🚀 学習の進め方

1. **環境構築**: Docker を使って学習環境を起動
2. **順番に学習**: CH1 から順番に進める（飛ばさないことを推奨）
3. **手を動かす**: 必ずコードを自分で入力して実行する
4. **質問する**: 分からないことは AI エージェントに積極的に質問
5. **復習する**: 演習問題で理解度を確認

## 💡 AI エージェントの活用方法
