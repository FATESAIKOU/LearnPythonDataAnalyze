# CH4: NumPy 入門 - 数値計算の基礎

## 🎯 この章の目的

NumPy を使って効率的な数値計算ができるようになることを目指します。
NumPy の配列（ndarray）は Pandas や Matplotlib の基礎となるため、ここでしっかり理解しておくことが重要です。

## 📖 セクション構成

### Section1: NumPy 配列の基本
**目標**: NumPy 配列を作成し、基本的な属性を理解する

- NumPy とは何か
- import numpy as np
- np.array() で配列を作成
- 配列の属性（shape, dtype, ndim, size）
- 配列の作成関数（zeros, ones, arange, linspace）
- 乱数配列（np.random）

### Section2: 配列の操作
**目標**: 配列の要素にアクセスし、形状を変更できる

- インデックスで要素にアクセス（1次元・2次元）
- スライスで範囲を取得
- ブールインデックス（条件で抽出）
- reshape() で形状変更
- flatten(), ravel() で1次元化
- 配列の結合（concatenate, vstack, hstack）

### Section3: 配列の計算
**目標**: 配列を使った効率的な計算ができる

- 要素ごとの演算（+, -, *, /）
- ブロードキャスト（異なる形状の配列の計算）
- 集計関数（sum, mean, max, min, std）
- 軸（axis）を指定した集計
- ユニバーサル関数（np.sqrt, np.exp, np.log など）
- 行列演算の基礎（dot, @演算子）

## 💡 つまずきやすいポイント

| 問題 | 解決策 |
|------|--------|
| shape の理解が難しい | (行数, 列数) の順番。常に .shape で確認する習慣をつける |
| インデックスが範囲外 | Python は 0 から始まる。最後の要素は -1 でアクセス |
| 計算結果がおかしい | ブロードキャストのルールを確認。shape が合っているか確認 |
| 元の配列が変わった | スライスはビュー（参照）。コピーしたい場合は .copy() を使う |
