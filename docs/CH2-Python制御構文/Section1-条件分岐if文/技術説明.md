# Section1: 条件分岐（if文）

## 🎯 このセクションの目標

**条件によって異なる処理を実行できるようになる！**

「もし〜なら」という判断をプログラムで表現し、状況に応じた処理ができるようになりましょう。

---

## 🤔 条件分岐とは？

日常生活でもこんな判断をしていますよね：

- **もし**雨が降っていたら → 傘を持っていく
- **もし**テストで80点以上なら → 合格
- **もし**20歳以上なら → お酒を買える

プログラムでも同じように「条件によって処理を変える」ことができます。

---

## 🔍 比較演算子

まず、条件を作るための「比較演算子」を学びましょう。

| 演算子 | 意味 | 例 | 結果 |
|--------|------|-----|------|
| `==` | 等しい | `5 == 5` | `True` |
| `!=` | 等しくない | `5 != 3` | `True` |
| `<` | より小さい | `3 < 5` | `True` |
| `>` | より大きい | `5 > 3` | `True` |
| `<=` | 以下 | `5 <= 5` | `True` |
| `>=` | 以上 | `5 >= 3` | `True` |

### 比較の結果は True か False

```python
print(10 == 10)  # True
print(10 == 5)   # False
print(10 > 5)    # True
print(10 < 5)    # False
```

## 📝 if 文の基本

### 構文

```python
if 条件:
    条件が True のときに実行する処理
```

**重要**: 処理の前には**インデント（スペース4つ）**が必要です！

### 例：点数が80点以上なら合格

```python
score = 85

if score >= 80:
    print("合格です！")
```

**実行結果：**
```
合格です！
```

### 条件が False の場合

```python
score = 70

if score >= 80:
    print("合格です！")

print("プログラム終了")
```

**実行結果：**
```
プログラム終了
```

`score >= 80` が `False` なので、`print("合格です！")` は実行されません。

---

## 🔀 if-else 文

条件が False のときの処理も書きたい場合は `else` を使います。

### 構文

```python
if 条件:
    条件が True のときの処理
else:
    条件が False のときの処理
```

### 例：合格・不合格の判定

```python
score = 70

if score >= 80:
    print("合格です！")
else:
    print("不合格です...")
```

**実行結果：**
```
不合格です...
```

---

## 🔀 if-elif-else 文

3つ以上の条件分岐には `elif`（else if の略）を使います。

### 構文

```python
if 条件1:
    条件1が True のときの処理
elif 条件2:
    条件2が True のときの処理
elif 条件3:
    条件3が True のときの処理
else:
    どの条件も True でないときの処理
```

### 例：成績の判定

```python
score = 75

if score >= 90:
    print("評価: A")
elif score >= 80:
    print("評価: B")
elif score >= 70:
    print("評価: C")
elif score >= 60:
    print("評価: D")
else:
    print("評価: F")
```

**実行結果：**
```
評価: C
```

**ポイント**: 上から順に条件をチェックし、最初に True になったところだけ実行されます。

## 🔗 論理演算子

複数の条件を組み合わせたいときは「論理演算子」を使います。

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `and` | かつ（両方 True） | `x > 0 and x < 10` |
| `or` | または（どちらか True） | `x < 0 or x > 10` |
| `not` | 否定（True と False を反転） | `not x == 0` |

### and の例

```python
age = 25
has_license = True

if age >= 18 and has_license:
    print("運転できます")
else:
    print("運転できません")
```

**実行結果：**
```
運転できます
```

### or の例

```python
day = "土曜日"

if day == "土曜日" or day == "日曜日":
    print("休日です")
else:
    print("平日です")
```

**実行結果：**
```
休日です
```

### not の例

```python
is_raining = False

if not is_raining:
    print("傘は不要です")
else:
    print("傘を持っていきましょう")
```

**実行結果：**
```
傘は不要です
```

## 📊 フローチャート

### if 文のフローチャート

```
    ┌─────────────┐
    │   開始      │
    └──────┬──────┘
           ↓
    ┌──────────────┐
    │ 条件を確認   │
    └──────┬───────┘
           ↓
      ┌────┴────┐
      │  条件   │
      └────┬────┘
       ┌───┴────┐
     True↓    ↓False
    ┌─────┐     │
    │処理A│     │
    └──┬──┘     │
       └──┬─────┘
          ↓
    ┌─────────────┐
    │    終了     │
    └─────────────┘
```

### if-else 文のフローチャート

```
    ┌─────────────┐
    │   開始      │
    └──────┬──────┘
           ↓
      ┌────┴────┐
      │  条件   │
      └────┬────┘
       ┌───┴────┐
     True↓    ↓False
    ┌─────┐  ┌─────┐
    │処理A│  │処理B│
    └──┬──┘  └──┬──┘
       └──┬─────┘
          ↓
    ┌─────────────┐
    │    終了     │
    └─────────────┘
```

## 🪆 ネストした if 文

if 文の中に if 文を書くこともできます。

```python
age = 20
is_member = True

if age >= 18:
    print("18歳以上です")
    if is_member:
        print("会員割引が適用されます")
    else:
        print("通常料金です")
else:
    print("18歳未満です")
```

**実行結果：**
```
18歳以上です
会員割引が適用されます
```

## 📝 文字列の比較

文字列も比較できます。

```python
password = "secret123"

if password == "secret123":
    print("ログイン成功！")
else:
    print("パスワードが違います")
```

### 大文字・小文字は区別される

```python
name = "Python"

print(name == "Python")  # True
print(name == "python")  # False
print(name == "PYTHON")  # False
```

## 🔢 in 演算子

リストや文字列に含まれているか確認できます。

```python
# リストでの使用
fruits = ["りんご", "バナナ", "みかん"]

if "りんご" in fruits:
    print("りんごがあります")

# 文字列での使用
message = "Hello, Python!"

if "Python" in message:
    print("Python が含まれています")
```

## ⚠️ よくあるエラー

### エラー例 1: = と == の間違い

```python
x = 10

# ❌ 間違い（代入になってしまう）
if x = 10:
    print("10です")

# ✅ 正しい（比較）
if x == 10:
    print("10です")
```

### エラー例 2: インデント忘れ

```python
score = 85

# ❌ 間違い
if score >= 80:
print("合格")  # IndentationError!

# ✅ 正しい
if score >= 80:
    print("合格")
```

### エラー例 3: コロン忘れ

```python
# ❌ 間違い
if score >= 80
    print("合格")  # SyntaxError!

# ✅ 正しい
if score >= 80:
    print("合格")
```
