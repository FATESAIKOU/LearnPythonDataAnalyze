## 演習3: 合計が50を超えるまで

### 問題

1から順に足していき、合計が50を超えたところで終了してください。

### 期待される出力例

```
1を足しました。合計: 1
2を足しました。合計: 3
3を足しました。合計: 6
...
10を足しました。合計: 55
合計が50を超えました！
```

### 解答例

```python
total = 0
num = 1

while total <= 50:
    total = total + num
    print(str(num) + "を足しました。合計: " + str(total))
    num = num + 1

print("合計が50を超えました！")
```

## 演習4: break の練習

### 問題

1から100までの数を表示し、7の倍数が見つかったらそこで終了してください。

### 解答例

```python
num = 1

while num <= 100:
```

```
6
```

## 演習6: 無限ループと break

### 問題

ユーザーが「quit」と入力するまで、入力を受け付け続けるプログラムを作成してください。

### 実行例

```
入力してください: hello
あなたの入力: hello
入力してください: python
あなたの入力: python
入力してください: quit
終了します
```

### 解答例

```python
while True:
    text = input("入力してください: ")
    if text == "quit":
        print("終了します")
        break
    print("あなたの入力: " + text)
```

## 演習7: 入力の検証

### 問題

1から10までの数字を入力してもらい、範囲外なら再入力を求めるプログラムを作成してください。

### 実行例

```
1〜10の数字を入力: 15
範囲外です。もう一度入力してください。
1〜10の数字を入力: 0
範囲外です。もう一度入力してください。
1〜10の数字を入力: 7
あなたの入力は 7 です
```

### 解答例

```python
while True:
    num = int(input("1〜10の数字を入力: "))
    if 1 <= num <= 10:
        break
    print("範囲外です。もう一度入力してください。")

print("あなたの入力は " + str(num) + " です")
```

## 演習8: パスワード認証（試行回数制限付き）

### 問題

パスワード「python」を3回まで試行できるプログラムを作成してください。

### 実行例

```
パスワードを入力: hello
間違いです。残り2回
パスワードを入力: world
間違いです。残り1回
パスワードを入力: python
ログイン成功！
```

### 解答例

```python
correct_password = "python"
attempts = 3

while attempts > 0:
    password = input("パスワードを入力: ")
    
    if password == correct_password:
        print("ログイン成功！")
        break
    else:
        attempts = attempts - 1
        if attempts > 0:
            print("間違いです。残り" + str(attempts) + "回")
        else:
            print("ロックされました")
```

## 演習9: 2のべき乗

### 問題

2のべき乗を表示し、1000を超えたら終了してください。

### 期待される出力

```
2^1 = 2
2^2 = 4
2^3 = 8
2^4 = 16
2^5 = 32
2^6 = 64
2^7 = 128
2^8 = 256
2^9 = 512
2^10 = 1024
1000を超えました！
```

### 解答例

```python
n = 1
result = 2

while result <= 1000:
    print("2^" + str(n) + " = " + str(result))
    n = n + 1
    result = 2 ** n

print("1000を超えました！")
```

## 演習10: while-else

### 問題

リスト内で「Python」を探すプログラムを作成してください。見つかったら「見つかりました」、見つからなければ「見つかりませんでした」と表示。

```python
languages = ["Java", "JavaScript", "Ruby", "Go", "Rust"]
```

### 解答例

```python
languages = ["Java", "JavaScript", "Ruby", "Go", "Rust"]
index = 0

while index < len(languages):
    if languages[index] == "Python":
        print("見つかりました！")
        break
    index = index + 1
else:
    print("見つかりませんでした")
```
