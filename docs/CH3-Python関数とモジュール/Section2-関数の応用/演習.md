## 演習1: デフォルト引数

### 問題

挨拶を表示する関数 `greet` を作成してください。
名前は必須、メッセージはデフォルトで「こんにちは」とします。

### 期待される動作

```python
greet("田中")
greet("佐藤", "おはよう")
```

### 期待される出力

```
こんにちは、田中さん！
おはよう、佐藤さん！
```

<details>
<summary>解答例を見る</summary>

```python
def greet(name, message="こんにちは"):
    print(message + "、" + name + "さん！")

greet("田中")
greet("佐藤", "おはよう")
```

</details>

## 演習2: 複数のデフォルト引数

### 問題

商品情報を表示する関数 `show_product` を作成してください。

- name: 商品名（必須）
- price: 価格（デフォルト: 0）
- stock: 在庫数（デフォルト: "不明"）

### 期待される動作

```python
show_product("りんご")
show_product("みかん", 150)
show_product("バナナ", 200, 50)
```

### 期待される出力

```
商品: りんご / 価格: 0円 / 在庫: 不明
商品: みかん / 価格: 150円 / 在庫: 不明
商品: バナナ / 価格: 200円 / 在庫: 50個
```

<details>
<summary>解答例を見る</summary>

```python
def show_product(name, price=0, stock="不明"):
    stock_str = stock if stock == "不明" else str(stock) + "個"
    print("商品: " + name + " / 価格: " + str(price) + "円 / 在庫: " + stock_str)

show_product("りんご")
show_product("みかん", 150)
show_product("バナナ", 200, 50)
```

</details>

## 演習4: *args（可変長位置引数）

### 問題

任意の数の数値を受け取って、その合計と平均を表示する関数 `calc_stats` を作成してください。

### 期待される動作

```python
calc_stats(10, 20, 30)
calc_stats(5, 10, 15, 20, 25)
```

### 期待される出力

```
合計: 60, 平均: 20.0
合計: 75, 平均: 15.0
```

<details>
<summary>解答例を見る</summary>

```python
def calc_stats(*numbers):
    total = sum(numbers)
    average = total / len(numbers)
    print("合計: " + str(total) + ", 平均: " + str(average))

calc_stats(10, 20, 30)
calc_stats(5, 10, 15, 20, 25)
```

</details>

## 演習5: *args で最大・最小を求める

### 問題

任意の数の数値を受け取って、最大値と最小値を返す関数 `find_min_max` を作成してください。

### 期待される動作

```python
min_val, max_val = find_min_max(3, 7, 2, 9, 4)
print("最小:", min_val)
print("最大:", max_val)
```

### 期待される出力

```
最小: 2
最大: 9
```

<details>
<summary>解答例を見る</summary>

```python
def find_min_max(*numbers):
    min_val = numbers[0]
    max_val = numbers[0]
    
    for num in numbers:
        if num < min_val:
            min_val = num
        if num > max_val:
            max_val = num
    
    return min_val, max_val

min_val, max_val = find_min_max(3, 7, 2, 9, 4)
print("最小:", min_val)
print("最大:", max_val)
```

</details>

## 演習6: **kwargs（可変長キーワード引数）

### 問題

任意のキーワード引数を受け取って、「キー: 値」の形式で表示する関数 `show_info` を作成してください。

### 期待される動作

```python
show_info(name="田中", age=25)
show_info(product="りんご", price=150, stock=30)
```

### 期待される出力

```
name: 田中
age: 25
---
product: りんご
price: 150
stock: 30
---
```

<details>
<summary>解答例を見る</summary>

```python
def show_info(**kwargs):
    for key, value in kwargs.items():
        print(str(key) + ": " + str(value))
    print("---")

show_info(name="田中", age=25)
show_info(product="りんご", price=150, stock=30)
```

</details>

## 演習8: ラムダ式でソート

### 問題

以下の果物リストを、価格の安い順にソートしてください。
ラムダ式を使って `sorted()` のキーを指定します。

```python
fruits = [
    {"name": "りんご", "price": 150},
    {"name": "バナナ", "price": 100},
    {"name": "ぶどう", "price": 300},
    {"name": "みかん", "price": 80}
]
```

### 期待される出力

```
みかん: 80円
バナナ: 100円
りんご: 150円
ぶどう: 300円
```

<details>
<summary>解答例を見る</summary>

```python
fruits = [
    {"name": "りんご", "price": 150},
    {"name": "バナナ", "price": 100},
    {"name": "ぶどう", "price": 300},
    {"name": "みかん", "price": 80}
]

sorted_fruits = sorted(fruits, key=lambda x: x["price"])

for fruit in sorted_fruits:
    print(fruit["name"] + ": " + str(fruit["price"]) + "円")
```

</details>

## 演習10: filter() とラムダ式

### 問題

リストから3文字以上の単語だけを抽出するプログラムを、`filter()` とラムダ式を使って書いてください。

words = ["a", "cat", "dog", "hi", "python", "go"]

## 演習11: スコープの理解

### 問題

以下のコードの出力を予想してから実行してください。

```python
x = 100

def my_function():
    x = 10
    print("関数内:", x)

my_function()
print("関数外:", x)
```

### 質問

1. 関数内の x と関数外の x は同じ変数ですか？
2. 関数を実行した後、x の値は変わりますか？

<details>
<summary>解答を見る</summary>

**出力:**
```
関数内: 10
関数外: 100
```

**解答:**
1. 別の変数です。関数内の x はローカル変数、関数外の x はグローバル変数です。
2. 変わりません。関数内で x に代入しても、それはローカル変数への代入なので、グローバル変数には影響しません。

</details>

## 演習12: global キーワード

### 問題

カウンターを実装してください。
`increment()` を呼ぶたびに `count` が1増えるようにします。

### 期待される動作

```python
count = 0

# ここに increment 関数を定義

increment()
increment()
increment()
print("カウント:", count)
```

### 期待される出力

```
カウント: 3
```

<details>
<summary>解答例を見る</summary>

```python
count = 0

def increment():
    global count
    count = count + 1

increment()
increment()
increment()
print("カウント:", count)
```

</details>
