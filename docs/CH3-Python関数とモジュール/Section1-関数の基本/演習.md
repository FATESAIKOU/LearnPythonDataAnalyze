## 演習8: docstring を書く

### 問題

円の面積を計算する関数 `circle_area` を作成してください。
docstring で引数と戻り値の説明を書いてください。

（円の面積 = π × 半径²、π は 3.14159 を使用）

### 期待される動作

```python
area = circle_area(5)
print("半径5の円の面積:", area)
```

### 期待される出力

```
半径5の円の面積: 78.53975
```

<details>
<summary>解答例を見る</summary>

```python
def circle_area(radius):
    """
    円の面積を計算する
    
    Args:
        radius: 円の半径
    
    Returns:
        円の面積
    """
    pi = 3.14159
    return pi * radius * radius

area = circle_area(5)
print("半径5の円の面積:", area)
```

</details>

## 演習9: リストを処理する関数

### 問題

数値のリストを受け取って、最大値を返す関数 `find_max` を作成してください。
（組み込み関数 max() を使わずに、自分で実装する）

### 期待される動作

```python
numbers = [3, 7, 2, 9, 4]
maximum = find_max(numbers)
print("最大値:", maximum)
```

### 期待される出力

```
最大値: 9
```

<details>
<summary>解答例を見る</summary>

```python
def find_max(numbers):
    """リストの中の最大値を返す"""
    max_value = numbers[0]
    
    for num in numbers:
        if num > max_value:
            max_value = num
    
    return max_value

numbers = [3, 7, 2, 9, 4]
maximum = find_max(numbers)
print("最大値:", maximum)
```

</details>

## 演習10: 関数を組み合わせる

### 問題

以下の2つの関数を作成し、組み合わせて使ってください。

1. `celsius_to_fahrenheit(c)`: 摂氏を華氏に変換（華氏 = 摂氏 × 9/5 + 32）
2. `fahrenheit_to_celsius(f)`: 華氏を摂氏に変換（摂氏 = (華氏 - 32) × 5/9）

### 期待される動作

```python
# 摂氏 → 華氏
print("0°C =", celsius_to_fahrenheit(0), "°F")
print("100°C =", celsius_to_fahrenheit(100), "°F")

# 華氏 → 摂氏
print("32°F =", fahrenheit_to_celsius(32), "°C")
print("212°F =", fahrenheit_to_celsius(212), "°C")
```

### 期待される出力

```
0°C = 32.0 °F
100°C = 212.0 °F
32°F = 0.0 °C
212°F = 100.0 °C
```

<details>
<summary>解答例を見る</summary>

```python
def celsius_to_fahrenheit(c):
    """摂氏を華氏に変換する"""
    return c * 9 / 5 + 32

def fahrenheit_to_celsius(f):
    """華氏を摂氏に変換する"""
    return (f - 32) * 5 / 9

# 摂氏 → 華氏
print("0°C =", celsius_to_fahrenheit(0), "°F")
print("100°C =", celsius_to_fahrenheit(100), "°F")

# 華氏 → 摂氏
print("32°F =", fahrenheit_to_celsius(32), "°C")
print("212°F =", fahrenheit_to_celsius(212), "°C")
```

</details>
