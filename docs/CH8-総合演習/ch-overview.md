## 📖 セクション構成

### Section1: プロジェクトの進め方
**目標**: データ分析プロジェクトの進め方を理解する

- ミニ分析プロセス（5ステップ）
  1. 問いを立てる（目的の明確化）
  2. データを見る（データの把握）
  3. データを整える（クリーニング、変換）
  4. 集計・可視化する（分析の実行）
  5. 結論をまとめる（レポート作成）
- 分析の問いを立てる
- 仮説を立てて検証する
- 結果を伝えるコツ

### Section2: ミニプロジェクト①：売上分析
**目標**: レストランの売上データを分析し、ビジネス改善の提案ができる

**課題**:
- tips データセット（seaborn 組み込み）を分析
- 以下の問いに答える分析とグラフを作成：
  - 曜日によって売上に差はあるか？
  - ランチとディナーで客単価に違いはあるか？
  - 人数と合計金額にはどんな関係があるか？
  - 売上を伸ばすにはどうすればよいか？

**使用技術**:
- Pandas：データ読み込み、集計、クロス集計
- Matplotlib：棒グラフ、散布図
- 相関分析

### Section3: ミニプロジェクト②：ペンギン観測データ分析
**目標**: ペンギンの観測データを分析し、種類ごとの特徴を明らかにする

**課題**:
- penguins データセット（seaborn 組み込み）を分析
- 以下の問いに答える分析とグラフを作成：
  - 種類によって体の大きさに違いはあるか？
  - 種類ごとにどの島に生息しているか？
  - くちばしの形状で種類を区別できるか？
  - 種類を判別するのに有効な指標は何か？

**使用技術**:
- Pandas：データ読み込み、欠損値処理、集計
- Matplotlib：散布図（種類別色分け）、棒グラフ
- 相関分析、クロス集計

## 📊 使用するデータセット

この章では seaborn ライブラリに組み込まれているデータセットを使用します。
追加のダウンロードは不要で、Docker 環境ではすぐに使用できます。

### tips データセット（売上分析）
レストランのチップに関するデータ（244件）

| カラム | 説明 |
|--------|------|
| total_bill | 合計金額（$） |
| tip | チップ額（$） |
| sex | 支払者の性別 |
| smoker | 喫煙者かどうか |
| day | 曜日 |
| time | 時間帯（Lunch/Dinner） |
| size | 人数 |

### penguins データセット（ペンギン観測データ分析）
南極のペンギン3種類の体の測定データ（344件）

| カラム | 説明 |
|--------|------|
| species | 種類（Adelie, Chinstrap, Gentoo） |
| island | 島（Biscoe, Dream, Torgersen） |
| bill_length_mm | くちばしの長さ（mm） |
| bill_depth_mm | くちばしの深さ（mm） |
| flipper_length_mm | ひれの長さ（mm） |
| body_mass_g | 体重（g） |
| sex | 性別 |
