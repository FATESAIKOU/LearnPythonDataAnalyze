# CH1: Python 基礎 - 学習の動機づけ

## 🎯 なぜ Python の基礎を学ぶのか？

あなたは Excel や GAS（Google Apps Script）で、AI が生成したコードをコピー＆ペーストして使っていませんか？
「動いたから OK」で済ませていませんか？

Python の基礎を学ぶことで、**コードの意味がわかる**ようになります。
これは、AI とより効果的に協働するための第一歩です。

```
┌─────────────────────────────────────────────────────────────────┐
│                Python 基礎を学ぶ目的                            │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   🔤 コードが読める      →  AI の出力を理解できる              │
│                                                                 │
│   ✏️ 簡単な修正ができる  →  AI に頼らず微調整できる            │
│                                                                 │
│   🐛 エラーの意味がわかる →  自分で問題を解決できる            │
│                                                                 │
│   💬 的確に質問できる    →  AI への指示が上手くなる            │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 🔄 この章を学ぶと何が変わるのか？（As-Is / To-Be 比較）

### 変化1: コードの見え方が変わる

```
┌─────────────────────────────────────────────────────────────────────────┐
│  シナリオ: AI が生成したコードを見たとき                                │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   AI が生成したコード:                                                  │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │  price = 1000                                                   │   │
│   │  tax_rate = 0.1                                                 │   │
│   │  total = price * (1 + tax_rate)                                 │   │
│   │  print(f"税込価格: {total}円")                                  │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
├────────────────────────────────┬────────────────────────────────────────┤
│     As-Is（学習前）            │       To-Be（学習後）                  │
├────────────────────────────────┼────────────────────────────────────────┤
│                                │                                        │
│   😰 「何が書いてあるか        │   😊 「なるほど！」                    │
│       わからない...」          │                                        │
│                                │   price = 1000   ← 変数に値を代入     │
│   • 記号の意味が不明           │   tax_rate = 0.1 ← 税率を変数に保存   │
│   • なぜ動くかわからない       │   total = ...    ← 計算結果を保存     │
│   • 修正したら壊れそう         │   print(...)     ← 結果を表示         │
│                                │                                        │
│   → とりあえず                │    →「税率を 0.08 に変えれば          │
│     コピペして祈る 🙏          │       8% 計算になるな」と理解できる    │
│                                │                                        │
└────────────────────────────────┴────────────────────────────────────────┘
```

### 変化2: エラーへの対応が変わる

```
┌─────────────────────────────────────────────────────────────────────────┐
│  シナリオ: コードを実行したらエラーが出た                               │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   実行したコード:                                                       │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │  priice = 1000                                                  │   │
│   │  total = price * 1.1                                            │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
│   エラーメッセージ:                                                     │
│   ┌─────────────────────────────────────────────────────────────────┐   │
│   │  NameError: name 'price' is not defined                         │   │
│   └─────────────────────────────────────────────────────────────────┘   │
│                                                                         │
├────────────────────────────────┬────────────────────────────────────────┤
│     As-Is（学習前）            │       To-Be（学習後）                  │
├────────────────────────────────┼────────────────────────────────────────┤
│                                │                                        │
│   😰 「エラー！怖い！」        │   😊 「あ、タイプミスだ」              │
│                                │                                        │
│   • 英語で何か言われてる...    │   NameError = 名前のエラー             │
│   • 意味がわからない           │   'price' is not defined               │
│   • AI にまた聞き直す          │      ↓                                │
│                                │   「price という変数が                 │
│   [あなた]→[AI]               │     定義されていない」                 │
│   「エラーが出ました、         │      ↓                                │
│     助けて」                   │   priice と打ち間違えてた！            │
│      ↓                        │      ↓                                │
│   (待ち時間...)                │   修正: priice → price ✅             │
│                                │                                        │
│   ⏰ 解決まで: 数分            │   ⏰ 解決まで: 数秒                    │
│                                │                                        │
└────────────────────────────────┴────────────────────────────────────────┘
```

### 変化3: Excel との対比で理解が深まる

```
┌─────────────────────────────────────────────────────────────────────────┐
│  シナリオ: 税込価格を計算したい                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│  📊 Excel での考え方:                                                   │
│  ┌─────────────────────────────────────────────────────────┐            │
│  │      A       │      B        │        C                 │            │
│  ├──────────────┼───────────────┼──────────────────────────┤            │
│  │    単価      │    税率       │       税込               │            │
│  ├──────────────┼───────────────┼──────────────────────────┤            │
│  │    1000      │     0.1       │   =A2*(1+B2)             │            │
│  │              │               │   → 1100                │            │
│  └──────────────┴───────────────┴──────────────────────────┘            │
│                                                                         │
│     セル A2 に「1000」を入れる                                          │
│     セル B2 に「0.1」を入れる                                           │
│     セル C2 に「=A2*(1+B2)」と入力 → 結果が表示される                  │
│                                                                         │
│  ─────────────────────────────────────────────────────────────────────  │
│                                                                         │
│  🐍 Python での考え方:                                                  │
│  ┌─────────────────────────────────────────────────────────────────┐    │
│  │  price = 1000           # セル A2 に値を入れるのと同じ          │    │
│  │  tax_rate = 0.1         # セル B2 に値を入れるのと同じ          │    │
│  │  total = price * (1 + tax_rate)  # C2 の式と同じ                │    │
│  │  print(total)           # → 1100.0                             │    │
│  └─────────────────────────────────────────────────────────────────┘    │
│                                                                         │
│     price という「箱」に 1000 を入れる                                  │
│     tax_rate という「箱」に 0.1 を入れる                                │
│     計算結果を total という「箱」に入れる                               │
│     print() で「箱」の中身を表示する                                    │
│                                                                         │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                         │
│   💡 ポイント: Excel の「セル」= Python の「変数」                      │
│                                                                         │
│   ┌──────────────┐          ┌─────────────┐                             │
│   │   Excel      │          │   Python    │                             │
│   ├──────────────┤          ├─────────────┤                             │
│   │ セル A2      │    ≒     │ 変数 price  │                             │
│   │ に値を入れる │          │ に値を代入  │                             │
│   └──────────────┘          └─────────────┘                             │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```
