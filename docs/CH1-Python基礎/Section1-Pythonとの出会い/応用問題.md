## 応用問題1: 自己紹介プログラム

### 問題

以下の情報を表示する自己紹介プログラムを作成してください：

- 名前
- 年齢
- 住んでいる場所
- 好きな食べ物
- Python を学ぶ理由

### 出力例

```
===== 自己紹介 =====
名前: 山田太郎
年齢: 25歳
住所: 東京都
好きな食べ物: ラーメン
Python を学ぶ理由: データ分析をしたい！
====================
```

### ヒント

- 区切り線も `print()` で表示できます
- `=` を20個並べるには `print("====================")` と書きます

<details>
<summary>解答例を見る</summary>

```python
print("===== 自己紹介 =====")
print("名前: 山田太郎")
print("年齢: 25歳")
print("住所: 東京都")
print("好きな食べ物: ラーメン")
print("Python を学ぶ理由: データ分析をしたい！")
print("====================")
```

</details>

## 応用問題2: アスキーアート

### 問題

`print()` を使って、以下のような簡単な顔を表示してください：

```

 ~~~
```

### ヒント

- 1行ずつ `print()` で表示します
- スペースも重要です！

<details>
<summary>解答例を見る</summary>

```python
print(" ^_^")

print(" ~~~")
```

</details>

## 応用問題3: 計算結果の表示

### 問題

以下の計算を行い、計算式と答えを両方表示してください：

1. 15 + 27
2. 100 - 37
3. 8 × 12
4. 144 ÷ 12

### 出力例

```
15 + 27 = 42
100 - 37 = 63
8 × 12 = 96
144 ÷ 12 = 12.0
```

### ヒント

- 計算式の部分は文字列として書きます
- 答えの部分は別の `print()` で計算結果を表示します
- または、1行にまとめる方法もあります（後の章で学びます）

<details>
<summary>解答例を見る</summary>

```python
# 方法1: 式と答えを別々に表示
print("15 + 27 =")
print(15 + 27)

print("100 - 37 =")
print(100 - 37)

print("8 × 12 =")
print(8 * 12)

print("144 ÷ 12 =")
print(144 / 12)
```

</details>

## 応用問題4: 買い物計算

### 問題

以下の買い物の合計金額を計算して表示してください：

| 商品 | 値段 |
|------|------|
| りんご | 150円 |
| パン | 200円 |
| 牛乳 | 180円 |
| お菓子 | 120円 |

### 出力例

```
===== 買い物リスト =====
りんご: 150円
パン: 200円
牛乳: 180円
お菓子: 120円
------------------------
合計: 650円
```

### ヒント

- 合計は `print(150 + 200 + 180 + 120)` で計算できます

<details>
<summary>解答例を見る</summary>

```python
print("===== 買い物リスト =====")
print("りんご: 150円")
print("パン: 200円")
print("牛乳: 180円")
print("お菓子: 120円")
print("------------------------")
print("合計:")
print(150 + 200 + 180 + 120)
```

</details>
