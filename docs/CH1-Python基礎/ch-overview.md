# CH1: Python 基礎 - はじめの一歩

## 🎯 この章の目的

プログラミング未経験者が Python の基本を理解し、簡単なコードを書けるようになることを目指します。
「プログラミングって難しそう...」という不安を払拭し、「自分でもできる！」という自信を持ってもらうことが最大の目標です。

---

## 📚 教授技術

| セクション | トピック | 学習内容 |
|------------|----------|----------|
| Section1 | Python との出会い | Jupyter Notebook の使い方、最初のプログラム |
| Section2 | 変数とデータ型 | 変数の作成、数値・文字列・真偽値 |
| Section3 | 基本的な演算 | 算術演算、文字列操作、型変換 |

---

## 🎓 教授方法

### アプローチ
- **即座に動かす**: 説明は最小限にして、すぐにコードを実行
- **成功体験優先**: まず「動いた！」を体験してから詳細を理解
- **エラーを恐れない**: エラーメッセージの読み方も学習の一部

### 進め方
1. 各セクションの「技術説明.md」を読む
2. 説明内のコードを Jupyter Notebook で実行
3. 「演習.md」の問題を解く
4. 余裕があれば「応用問題.md」にチャレンジ

---

## 📖 セクション構成

### Section1: Python との出会い
**目標**: Jupyter Notebook を使って最初のプログラムを実行する

- Jupyter Notebook の基本操作
- セルの実行方法
- `print()` 関数で文字を表示
- コメントの書き方

### Section2: 変数とデータ型
**目標**: データを変数に保存して使えるようになる

- 変数とは何か
- 変数の命名ルール
- 数値型（int, float）
- 文字列型（str）
- 真偽値型（bool）
- `type()` 関数でデータ型を確認

### Section3: 基本的な演算
**目標**: 変数を使って計算や文字列操作ができる

- 算術演算子（+, -, *, /, //, %, **）
- 文字列の連結と繰り返し
- 型変換（int(), float(), str()）
- 入力の受け取り（input()）

## 💡 つまずきやすいポイント

| 問題 | 解決策 |
|------|--------|
| セルが実行できない | Shift + Enter を押す、またはセルが選択されているか確認 |
| 変数名でエラー | 数字から始めていないか、予約語を使っていないか確認 |
| 文字列でエラー | クォーテーション（' または "）で囲んでいるか確認 |
| 計算結果がおかしい | 整数除算（//）と通常の除算（/）の違いを確認 |
