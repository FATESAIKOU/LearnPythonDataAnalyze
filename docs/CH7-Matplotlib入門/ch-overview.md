# CH7: Matplotlib 入門 - データの可視化

## 🎯 この章の目的

Matplotlib を使ってデータをグラフで可視化できるようになることを目指します。
分析結果を視覚的に伝えることで、データから得られた洞察をより効果的に共有できるようになります。

## 📖 セクション構成

### Section1: Matplotlib の基本
**目標**: Matplotlib の基本構造を理解し、簡単なグラフを描ける

- Matplotlib とは何か
- import matplotlib.pyplot as plt
- Jupyter Notebook での表示（%matplotlib inline）
- Figure と Axes の概念
- plt.plot() で折れ線を描く
- plt.show() でグラフ表示
- 日本語表示の設定（japanize-matplotlib）

### Section2: 折れ線グラフと散布図
**目標**: 時系列データや相関関係を可視化できる

- 折れ線グラフ（plot）の詳細オプション
- 線のスタイル（色、太さ、種類、マーカー）
- 複数の線を重ねて描画
- 散布図（scatter）の基本
- 散布図のオプション（サイズ、色、透明度）
- 相関関係の可視化

### Section3: 棒グラフとヒストグラム
**目標**: カテゴリ比較や分布を可視化できる

- 棒グラフ（bar）の基本
- 横棒グラフ（barh）
- 積み上げ棒グラフ
- グループ化された棒グラフ
- ヒストグラム（hist）の基本
- ビン数の調整
- 円グラフ（pie）の基本

### Section4: グラフの装飾
**目標**: 見やすく美しいグラフを作成できる

- タイトルの追加（title）
- 軸ラベルの追加（xlabel, ylabel）
- 凡例の追加（legend）
- 軸の範囲設定（xlim, ylim）
- 目盛りのカスタマイズ（xticks, yticks）
- グリッド線の追加（grid）
- 色の指定方法（名前、16進数、RGB）
- スタイルシートの使用（plt.style.use）

### Section5: 複数グラフとエクスポート
**目標**: 複数のグラフを配置し、画像として保存できる

- サブプロット（subplot）の基本
- plt.subplots() での作成
- グラフの配置とサイズ調整
- 画像として保存（savefig）
- 保存形式（PNG, PDF, SVG）
- 解像度の設定（dpi）
- Pandas との連携（df.plot）

## 💡 つまずきやすいポイント

| 問題 | 解決策 |
|------|--------|
| グラフが表示されない | Jupyter では %matplotlib inline を実行。または plt.show() を呼ぶ |
| 日本語が文字化け | japanize-matplotlib をインポート |
| 凡例が表示されない | plot() に label パラメータを設定し、legend() を呼ぶ |
| グラフが重なる | plt.figure() で新しい Figure を作成、または plt.clf() でクリア |
