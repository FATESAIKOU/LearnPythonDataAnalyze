# Python データ分析入門

プログラミング未経験者向けの Python データ分析教材へようこそ！

## 🎯 この教材について

本教材は、**Excel やスプレッドシートの操作経験がある方**が、Python を使ったデータ分析の基礎を習得することを目的としています。

## 📚 学習内容

| 章 | テーマ | 内容 |
|---|--------|------|
| CH1 | Python 基礎 | 変数、データ型、基本操作 |
| CH2 | 制御構文 | if文、for文、while文 |
| CH3 | 関数とモジュール | 関数定義、ライブラリ活用 |
| CH4 | NumPy 入門 | 配列操作、数値計算 |
| CH5 | Pandas 入門 | Series、DataFrame、データ操作 |
| CH6 | Pandas 応用 | クリーニング、結合、集計 |
| CH7 | Matplotlib 入門 | グラフ作成、可視化 |
| CH8 | 総合演習 | 実践プロジェクト |

## 🚀 始め方

1. 左のサイドバーから **CH1 Python基礎** を選択
2. 各セクションの **技術説明** → **演習** → **応用問題** の順に進める
3. 分からないことは AI エージェントに質問！

## 💻 学習環境

Docker を使った JupyterLab 環境を提供しています。詳しくは [教材概要](overview.md) をご覧ください。
