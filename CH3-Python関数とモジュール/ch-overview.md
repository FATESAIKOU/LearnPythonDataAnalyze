# CH3: Python 関数とモジュール - コードを整理する

## 🎯 この章の目的

コードを再利用可能な部品として整理し、効率的にプログラムを書けるようになることを目指します。
関数を作成して処理をまとめたり、便利なモジュールを活用したりする方法を学びます。

---

## 📚 教授技術

| セクション | トピック | 学習内容 |
|------------|----------|----------|
| Section1 | 関数の基本 | 関数の定義、引数、戻り値 |
| Section2 | 関数の応用 | デフォルト引数、可変長引数、ラムダ式 |
| Section3 | モジュールとパッケージ | import、標準ライブラリ、外部ライブラリ |

---

## 🎓 教授方法

### アプローチ
- **DRY原則**: Don't Repeat Yourself（繰り返しを避ける）を意識
- **実用例から学ぶ**: 実際に便利な関数を作りながら理解
- **ライブラリ体験**: NumPy/Pandas への橋渡しとして標準ライブラリを体験

### 進め方
1. 各セクションの「技術説明.md」を読む
2. 説明内のコードを実行して動作を確認
3. 「演習.md」で自分で関数を作成
4. 「応用問題.md」で実践的な関数設計にチャレンジ

## 📖 セクション構成

### Section1: 関数の基本
**目標**: 自分で関数を定義して呼び出せるようになる

- 関数とは何か（なぜ必要か）
- def キーワードで関数を定義
- 引数（パラメータ）の使い方
- return で値を返す
- ドキュメンテーション文字列（docstring）

### Section2: 関数の応用
**目標**: より柔軟な関数を作成できるようになる

- デフォルト引数
- キーワード引数
- 可変長引数（*args, **kwargs）
- ラムダ式（lambda）
- スコープ（ローカル変数とグローバル変数）

### Section3: モジュールとパッケージ
**目標**: 外部のコードを活用できるようになる

- import 文の使い方
- from ... import ... 構文
- as でエイリアスを付ける
- 標準ライブラリ（math, random, datetime）
- pip でパッケージをインストール
- NumPy/Pandas/Matplotlib の紹介

## 💡 つまずきやすいポイント

| 問題 | 解決策 |
|------|--------|
| 関数が実行されない | 関数は定義しただけでは実行されない。呼び出しを忘れていないか確認 |
| return がない | return を書き忘れると None が返される |
| import エラー | モジュール名のスペルミス、またはインストールされていないか確認 |
| 変数が見つからない | スコープの問題。関数内で定義した変数は関数外では使えない |
