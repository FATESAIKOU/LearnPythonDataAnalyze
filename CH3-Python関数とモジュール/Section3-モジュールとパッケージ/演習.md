## 演習1: math モジュールの基本

### 問題

math モジュールを使って、以下を計算してください。

1. 25 の平方根
2. 2 の 8 乗
3. 円周率（π）の値

### 期待される出力

```
25の平方根: 5.0
2の8乗: 256.0
円周率: 3.141592653589793
```

<details>
<summary>解答例を見る</summary>

```python
import math

print("25の平方根:", math.sqrt(25))
print("2の8乗:", math.pow(2, 8))
print("円周率:", math.pi)
```

</details>

## 演習2: from ... import の使い方

### 問題

math モジュールから `floor` と `ceil` だけをインポートして、以下を計算してください。

1. 3.7 を切り捨て
2. 3.2 を切り上げ

### 期待される出力

```
3.7の切り捨て: 3
3.2の切り上げ: 4
```

<details>
<summary>解答例を見る</summary>

```python
from math import floor, ceil

print("3.7の切り捨て:", floor(3.7))
print("3.2の切り上げ:", ceil(3.2))
```

</details>

## 演習3: random モジュールで乱数

### 問題

random モジュールを使って、以下を実行してください。

1. 1〜100 の間のランダムな整数を3つ表示
2. リスト `["赤", "青", "緑", "黄"]` からランダムに1つ選ぶ

### 期待される出力（例）

```
ランダムな数: 42 87 15
選ばれた色: 青
```

<details>
<summary>解答例を見る</summary>

```python
import random

# ランダムな整数を3つ
n1 = random.randint(1, 100)
n2 = random.randint(1, 100)
n3 = random.randint(1, 100)
print("ランダムな数:", n1, n2, n3)

# リストからランダムに選ぶ
colors = ["赤", "青", "緑", "黄"]
selected = random.choice(colors)
print("選ばれた色:", selected)
```

</details>

## 演習4: datetime モジュールで日付

### 問題

datetime モジュールを使って、以下を表示してください。

1. 今日の日付
2. 7日後の日付
3. 自分の誕生日（適当な日付でOK）

### 期待される出力（例）

```
今日: 2024-01-15
7日後: 2024-01-22
誕生日: 1990-05-15
```

<details>
<summary>解答例を見る</summary>

```python
import datetime

# 今日の日付
today = datetime.date.today()
print("今日:", today)

# 7日後
next_week = today + datetime.timedelta(days=7)
print("7日後:", next_week)

# 誕生日
birthday = datetime.date(1990, 5, 15)
print("誕生日:", birthday)
```

</details>

## 演習5: as でエイリアス

### 問題

datetime モジュールを `dt` というエイリアスでインポートして、現在の時刻を表示してください。

### 期待される出力（例）

```
現在時刻: 2024-01-15 10:30:45
```

<details>
<summary>解答例を見る</summary>


## 演習8: 複数モジュールの組み合わせ（日付と乱数）

### 問題

datetime と random モジュールを使って、「今日からランダムな日数後の日付」を表示してください。

1. 1〜30 の間でランダムな日数を生成
2. 今日からその日数後の日付を計算

### 期待される出力（例）

```
今日: 2024-01-15
ランダムな日数: 17日後
その日付: 2024-02-01
```

<details>
<summary>解答例を見る</summary>

```python
import datetime
import random

today = datetime.date.today()
days = random.randint(1, 30)
future = today + datetime.timedelta(days=days)

print("今日:", today)
print("ランダムな日数:", str(days) + "日後")
print("その日付:", future)
```

</details>

## 演習9: 関数とモジュールの組み合わせ

### 問題

random モジュールと関数を組み合わせて、「おみくじ」関数を作成してください。

**仕様:**
- 大吉、中吉、小吉、吉、末吉、凶 のどれかをランダムに返す

### 期待される動作

```python
print(omikuji())
print(omikuji())
print(omikuji())
```

### 期待される出力（例）

```
大吉
凶
中吉
```

<details>
<summary>解答例を見る</summary>

```python
import random

def omikuji():
    """おみくじを引く"""
    results = ["大吉", "中吉", "小吉", "吉", "末吉", "凶"]
    return random.choice(results)

print(omikuji())
print(omikuji())
print(omikuji())
```

</details>

## 演習10: 総合演習

### 問題

math, random, datetime を組み合わせて、以下のプログラムを作成してください。

1. 今日の日付を表示
2. 1〜100 のランダムな数を生成
3. その数の平方根を計算（小数点以下2桁まで）
4. 結果を「レポート」として表示

### 期待される出力（例）

```
===== レポート =====
日付: 2024-01-15
選ばれた数: 64
平方根: 8.00
====================
```

<details>
<summary>解答例を見る</summary>

```python
import math
import random
import datetime

# 今日の日付
today = datetime.date.today()

# ランダムな数
num = random.randint(1, 100)

# 平方根
sqrt_num = math.sqrt(num)

# レポート表示
print("===== レポート =====")
print("日付:", today)
print("選ばれた数:", num)
print("平方根:", format(sqrt_num, ".2f"))
print("====================")
```

</details>
