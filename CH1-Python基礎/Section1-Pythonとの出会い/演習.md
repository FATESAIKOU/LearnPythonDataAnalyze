## 演習3: 複数行の表示

### 問題

以下の3行を表示してください（3つの print 文を使います）：

```
私は Python を学んでいます
今日は最初の日です
頑張ります！
```

### 解答例

```python
print("私は Python を学んでいます")
print("今日は最初の日です")
print("頑張ります！")
```

### 確認ポイント
- [ ] 3行すべて表示されましたか？

## 演習4: 数字の表示

### 問題

以下の数字をそれぞれ表示してください：

1. `100`
2. `3.14159`
3. `-50`

### 解答例

```python
print(100)
print(3.14159)
print(-50)
```

### 確認ポイント
- [ ] 数字はクォーテーションで囲まなくても表示できましたか？

## 演習5: 簡単な計算

### 問題

以下の計算結果を表示してください：

1. 5 + 3
2. 10 - 4
3. 6 × 7（Python では `*` を使います）
4. 20 ÷ 4（Python では `/` を使います）

### 解答例

```python
print(5 + 3)
print(10 - 4)
print(6 * 7)
print(20 / 4)
```

### 期待される出力

```
8
6
42
5.0
```

### 確認ポイント
- [ ] 4つの計算結果が正しく表示されましたか？
- [ ] 割り算の結果が `5.0` になっていることに気づきましたか？
