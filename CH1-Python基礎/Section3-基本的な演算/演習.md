## 演習1: 基本的な計算

### 問題

以下の計算を Python で実行してください：

1. 25 + 17
2. 100 - 43
3. 12 × 8
4. 56 ÷ 7

### 解答例

```python
print(25 + 17)   # 42
print(100 - 43)  # 57
print(12 * 8)    # 96
print(56 / 7)    # 8.0
```

### 確認ポイント
- [ ] 4つの計算結果が正しく表示されましたか？
- [ ] 割り算の結果が `8.0`（小数）になっていますか？

## 演習5: 計算の優先順位

### 問題

以下の計算結果を予想してから、実行して確認してください：

1. `2 + 3 * 4`
2. `(2 + 3) * 4`
3. `10 - 6 / 2`
4. `(10 - 6) / 2`

### 解答例

```python
print(2 + 3 * 4)    # 14
print((2 + 3) * 4)  # 20
print(10 - 6 / 2)   # 7.0
print((10 - 6) / 2) # 2.0
```

## 演習6: 文字列の連結

### 問題

以下の変数を使って、フルネームを作成してください：

```python
first_name = "山田"
last_name = "花子"
```

「山田 花子」（間にスペースあり）を表示してください。

### 解答例

```python
first_name = "山田"
last_name = "花子"

full_name = first_name + " " + last_name
print(full_name)  # 山田 花子
```

## 演習7: 文字列の繰り返し

### 問題

以下を表示してください：

1. `=` を30個並べた線
2. `*` を10個並べた線
3. `"Hello "` を5回繰り返し

### 解答例

## 演習8: 型変換（int）

### 問題

以下の変換を行ってください：

## 演習9: 型変換（str）

### 問題

以下の変数を使って、メッセージを作成してください：

```python
age = 30
height = 175.5
```

「私は30歳で、身長は175.5cmです」と表示してください。

### 解答例

```python
age = 30
height = 175.5

message = "私は" + str(age) + "歳で、身長は" + str(height) + "cmです"
print(message)
```

## 演習11: input() と計算

### 問題

ユーザーに年齢を入力してもらい、10年後の年齢を計算して表示してください。

### 出力例

```
年齢を入力してください: 25
10年後は35歳ですね！
```

### 解答例

```python
age = input("年齢を入力してください: ")
age = int(age)  # 文字列を整数に変換
future_age = age + 10
print("10年後は" + str(future_age) + "歳ですね！")
```

### 確認ポイント
- [ ] input() の結果を int() で変換しましたか？
- [ ] 結果を表示するときに str() で変換しましたか？

## 演習12: 複合代入演算子

### 問題

以下のコードを複合代入演算子を使って書き換えてください：

```python
# 元のコード
score = 100
score = score + 10
score = score - 5
score = score * 2
```

### 解答例

```python
score = 100
score += 10   # score = 110
print(score)

score -= 5    # score = 105
print(score)

score *= 2    # score = 210
print(score)
```
