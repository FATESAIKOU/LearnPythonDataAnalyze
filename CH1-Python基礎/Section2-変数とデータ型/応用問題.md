## 応用問題2: 変数の入れ替え

### 問題

2つの変数 `a` と `b` の値を入れ替えてください。

**開始状態：**
```python
a = 10
b = 20
```

**終了状態：**
```
a = 20
b = 10
```

### ヒント

一時的な変数を使う方法と、Python 特有の方法があります。

<details>
<summary>解答例を見る</summary>

```python
# 方法1: 一時変数を使う
a = 10
b = 20
print("入れ替え前:")
print(a)
print(b)

temp = a
a = b
b = temp

print("入れ替え後:")
print(a)
print(b)
```

```python
# 方法2: Python の便利な書き方
a = 10
b = 20
print("入れ替え前:")
print(a)
print(b)

a, b = b, a

print("入れ替え後:")
print(a)
print(b)
```

</details>

## 応用問題3: 型の違いを理解する

### 問題

以下のコードを実行して、結果を予想してから確認してください：

```python
num1 = 10
num2 = "10"

print(type(num1))
print(type(num2))
```

### 質問

1. `num1` と `num2` は同じですか？違いますか？
2. なぜ違うのでしょうか？

<details>
<summary>解答を見る</summary>

**結果：**
```
<class 'int'>
<class 'str'>
```

**解説：**
- `num1 = 10` → 数値の10（計算に使える）
- `num2 = "10"` → 文字列の"10"（文字として扱う）

クォーテーションで囲むと文字列になります！

</details>

## 応用問題5: 変数の連鎖

### 問題

以下のコードを実行して、結果を予想してください：

```python
a = 5
b = a
a = 10

print(a)
print(b)
```

### 質問

`b` の値は `5` ですか？それとも `10` ですか？

<details>
<summary>解答を見る</summary>

**結果：**
```
10
5
```

**解説：**
- `b = a` の時点で `b` には `5` がコピーされます
- その後 `a` を `10` に変更しても、`b` は影響を受けません
- 変数は「箱」なので、それぞれ独立しています

</details>

## 応用問題7: 変数の更新シミュレーション

### 問題

ゲームのシナリオをシミュレートしてください：

1. 初期 HP は `100`
2. 敵から `30` のダメージを受けた
3. 回復薬で `20` 回復した
4. 敵から `25` のダメージを受けた

各ステップで HP を表示してください。

### 出力例

```
初期HP: 100
敵からダメージ！ 現在HP: 70
回復！ 現在HP: 90
敵からダメージ！ 現在HP: 65
```

<details>
<summary>解答例を見る</summary>

```python
hp = 100
print("初期HP:")
print(hp)

hp = hp - 30
print("敵からダメージ！ 現在HP:")
print(hp)

hp = hp + 20
print("回復！ 現在HP:")
print(hp)

hp = hp - 25
print("敵からダメージ！ 現在HP:")
print(hp)
```

</details>
