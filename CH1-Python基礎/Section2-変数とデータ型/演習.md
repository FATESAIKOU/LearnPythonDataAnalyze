# Section2: 演習問題

変数とデータ型について学んだことを、実際にコードを書いて確認しましょう。

## 演習1: はじめての変数

### 問題

以下の手順で変数を作成し、表示してください：

1. `greeting` という変数に `"こんにちは"` を代入
2. `print()` で `greeting` を表示

### 期待される出力

```
こんにちは
```

### 解答例

```python
greeting = "こんにちは"
print(greeting)
```

### 確認ポイント
- [ ] 変数 `greeting` に文字列を代入できましたか？
- [ ] `print(greeting)` で中身が表示されましたか？

## 演習3: type() で型を確認

### 問題

以下の各値の型を `type()` で確認してください：

1. `42`
2. `3.14`
3. `"Python"`
4. `True`

### 解答例

```python
print(type(42))
print(type(3.14))
print(type("Python"))
print(type(True))
```

### 期待される出力

```
<class 'int'>
<class 'float'>
<class 'str'>
<class 'bool'>
```

### 確認ポイント
- [ ] 4つの型をすべて確認できましたか？

## 演習7: 真偽値（bool）の練習

### 問題

以下の変数を作成し、表示と型の確認をしてください：

1. `is_student` に `True` を代入
2. `has_car` に `False` を代入
3. 両方を表示
4. 両方の型を確認

```
True

<class 'bool'>
```

## 演習8: 変数の型を確認

### 問題

以下の変数を作成し、それぞれの型を確認してください：

```python
a = 100
b = 99.9
c = "100"
d = True
```

### 解答例

```python
a = 100
b = 99.9
c = "100"
d = True

print(type(a))  # int
print(type(b))  # float
print(type(c))  # str（文字列の "100"）
print(type(d))  # bool
```

### 確認ポイント
- [ ] `100` と `"100"` は違う型であることを理解しましたか？

## 演習10: 変数名の練習

### 問題

以下の情報を適切な変数名で保存してください：

1. ユーザーの名前 → ?
2. 商品の価格 → ?
3. ログイン状態 → ?
4. 生まれた年 → ?

```python
birth_year = 1995

print(user_name)
print(product_price)
print(is_logged_in)
print(birth_year)
```

### 確認ポイント
- [ ] 意味のわかる変数名をつけられましたか？
- [ ] スネークケース（単語をアンダースコアで区切る）を使いましたか？
