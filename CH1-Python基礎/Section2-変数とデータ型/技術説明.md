# Section2: 変数とデータ型

## 🎯 このセクションの目標

**データを変数に保存して、繰り返し使えるようになる！**

変数を使うと、データに名前をつけて保存できます。これはプログラミングの基本中の基本です。

---

## 📦 変数とは？

変数は「データを入れる箱」のようなものです。

```python
name = "太郎"
```

この例では：
- `name` → 箱の名前（変数名）
- `"太郎"` → 箱に入れるデータ（値）
- `=` → 「代入する」という意味

---

## 🚀 変数を使ってみよう

### Step 1: 変数にデータを入れる

```python
name = "太郎"
```

### Step 2: 変数の中身を表示する

```python
print(name)
```

**実行結果：**
```
太郎
```

変数名だけで、中身のデータを取り出せます！

---

## 💡 変数の便利さ

同じデータを何度も使うとき、変数が便利です。

### 変数を使わない場合

```python
print("田中さん、こんにちは！")
print("田中さん、今日の調子はどうですか？")
print("田中さん、また会いましょう！")
```

### 変数を使う場合

```python
name = "田中"
print(name + "さん、こんにちは！")
print(name + "さん、今日の調子はどうですか？")
print(name + "さん、また会いましょう！")
```

名前を変更したいとき、1箇所を変えるだけで OK！

---

## 📝 変数の命名ルール

変数名には以下のルールがあります：

### ✅ 使える文字

| 文字 | 例 |
|------|-----|
| アルファベット | `name`, `age`, `Price` |
| 数字（先頭以外） | `score1`, `data2023` |
| アンダースコア | `user_name`, `_private` |
| 日本語（非推奨） | `名前`, `年齢` |

### ❌ 使えない例

```python
# ❌ 数字から始まる
1st_place = "Gold"  # エラー！

# ❌ ハイフンを含む
user-name = "Taro"  # エラー！

# ❌ スペースを含む
user name = "Taro"  # エラー！

# ❌ 予約語
print = "hello"  # 避けるべき！
```

### 💡 おすすめの書き方

```python
# スネークケース（Python で推奨）
user_name = "太郎"
birth_year = 2000

# 意味のわかる名前をつける
age = 25          # ✅ 良い
a = 25            # ❌ 意味がわからない
```

---

## 🔢 データ型（Type）

Python のデータには「型」があります。

### 主なデータ型

| 型 | 名前 | 例 |
|----|------|-----|
| `int` | 整数 | `1`, `100`, `-50` |
| `float` | 小数 | `3.14`, `0.5`, `-2.5` |
| `str` | 文字列 | `"Hello"`, `'Python'` |
| `bool` | 真偽値 | `True`, `False` |

---

## 🔢 整数型（int）

小数点のない数値です。

```python
age = 25
year = 2024
temperature = -5

print(age)        # 25
print(year)       # 2024
print(temperature)  # -5
```

---

## 🔢 小数型（float）

小数点のある数値です。

```python
pi = 3.14159
height = 170.5
rate = 0.08

print(pi)      # 3.14159
print(height)  # 170.5
print(rate)    # 0.08
```

---

## 📝 文字列型（str）

文字のデータです。クォーテーションで囲みます。

```python
# シングルクォーテーション
name = 'Python'

# ダブルクォーテーション
message = "Hello, World!"

# どちらでも OK（統一するのがおすすめ）
```

### 文字列の中にクォーテーションを含める

```python
# ダブルの中にシングル
message = "It's a beautiful day!"

# シングルの中にダブル
quote = 'He said "Hello"'

print(message)  # It's a beautiful day!
print(quote)    # He said "Hello"
```

---

## ✅ 真偽値型（bool）

「はい」か「いいえ」の2つの値だけを持ちます。

```python
is_student = True
is_adult = False

print(is_student)  # True
print(is_adult)    # False
```

**注意：** `True` と `False` は大文字から始まります！

```python
# ❌ 間違い
is_ok = true   # エラー！

# ✅ 正しい
is_ok = True
```

---

## 🔍 type() 関数

データ型を確認するには `type()` 関数を使います。

```python
age = 25
print(type(age))
```

**実行結果：**
```
<class 'int'>
```

### いろいろな型を確認

```python
print(type(100))        # <class 'int'>
print(type(3.14))       # <class 'float'>
print(type("Hello"))    # <class 'str'>
print(type(True))       # <class 'bool'>
```

---

## 🔄 変数の値を変更する

変数は何度でも値を変更できます。

```python
score = 80
print(score)   # 80

score = 90
print(score)   # 90

score = 100
print(score)   # 100
```

---

## 📊 複数の変数を一度に作る

```python
# 別々に書く
x = 10
y = 20
z = 30

# まとめて書く
x, y, z = 10, 20, 30

# 同じ値を入れる
a = b = c = 0
```

---

## ⚠️ よくあるエラー

### エラー例 1: 未定義の変数

```python
print(message)  # ❌ message が定義されていない
```

**エラーメッセージ：**
```
NameError: name 'message' is not defined
```

**解決策：** 使う前に変数を定義する

```python
message = "Hello"
print(message)  # ✅ OK
```

### エラー例 2: スペルミス

```python
username = "Taro"
print(usernme)  # ❌ スペルミス！
```

**解決策：** 変数名を正確に書く
